\doxysubsection{Detailed Description}
Implementation of the Finite State Machine (\doxylink{class_f_s_m}{FSM}) controlling robot operation modes. 

This file defines the behavior of each operational mode of the Ping Pong Launcher Robot, including initialization, manual control, target aiming, and automatic LIDAR-\/based targeting.
